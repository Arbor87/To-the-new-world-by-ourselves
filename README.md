# To-the-new-world-by-ourselves
《致新世界》游戏说明：
由姜熙丹策划，赵林可欣编程，陈慧琳建模。
1.游戏背景：你被困在森林之中，通过吃豆和回答问题来推动剧情发展，最后打出4种不同的结局。
2.游戏玩法：玩家通过控制小人，进行吃豆，每吃满三个豆会有剧情的变化，同时游戏进行互动提问，游戏通过玩家的选择来决定游戏的走向和最终的结局。
3.游戏主题：基于集体温暖互动的主题，游戏将自我的原始封闭精神空间比拟做森林，为了走出森林，需要自己的选择和勇气，也对应着在现实之中，挑战自己，勇敢迈出社交的一步，体会到集体的温暖与力量。
就像游戏中，玩家通过拯不拯救他人（即自己）来决定自己是否可以走出自我的孤独，从而拥抱森林之外世界的温暖。
4.游戏名称：之所以选择新世界作为名字，是因为每一个人最开始都会拥有一个旧的闭塞的旧世界，我们将会通过与他人的互动和来自集体的温暖来融化自己有强烈防御心的外壳，从此以后，我们每走出的一步都是新世界。
5.巧思：1.游戏分别设置2个假结局和2个真结局。
真结局的含义是最后是否走出森林，对应着在现实中是否突破自我，融入集体。
假结局的含义是是否走出森林是未知数，可能固步自封，也可能通过自己的努力走出去，因为我们不能否定一个人前进就一定失败。
2.玩家遇见的ta实际上就是我们理想中完美的自己，拥有我们所渴望的一切，擅长社交，会带领我们一步一步走出阴霾。其中，当玩家想知道对方是谁时，屏幕全黑，我们可以在屏幕之中看见自己。
而正是我们自己要勇敢面对困难的森林，需要我们自己一步一步融入新世界之中。
